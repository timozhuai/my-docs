## 洛雪音乐助手 (LX Music)

一款开源免费的聚合型音乐软件，基于 Electron + Vue 3 开发，支持 Windows、macOS、Linux 平台。

### 核心特点

- **多平台聚合**：整合网易云、QQ音乐、酷狗、酷我、咪咕等12个平台
- **无损音质**：支持普通音质、高品质、无损APE/FLAC/WAV，最高 192kHz/24bit
- **开源免费**：代码公开，无广告，不收集隐私数据
- **便携模式**：支持 Windows 便携模式，数据本地存储

### 音源链接（推荐）

```
https://fastly.jsdelivr.net/gh/Huibq/keep-alive/render_api.js
```

### 音源导入方法

1. 打开洛雪音乐 → 设置 → 基本设置 → 音乐来源 → 自定义源管理
2. 点击「导入」或「添加」
3. 粘贴上面的音源链接，选择在线导入
4. 成功后在音乐来源中选择「Huibq_lxmusic」

### 下载地址

- [window桌面端](https://github.com/lyswhut/lx-music-desktop)
- [安卓移动端](https://github.com/lyswhut/lx-music-mobile)

## MusicFree

一款开源免费的音乐聚合软件，支持 Android、Windows、macOS、Linux 平台。

### 核心特点

- **插件化设计**：支持自定义音源插件
- **多平台聚合**：支持多种音乐平台
- **开源免费**：无广告，不收集隐私

### 音源链接

```
https://fastly.jsdelivr.net/gh/Huibq/keep-alive/Music_Free/myPlugins.json
```

### 导入方法

1. 打开 MusicFree → 设置 → 插件
2. 点击右上角「+」号
3. 选择「添加网络插件」
4. 粘贴上面的音源链接
5. **重要**：在设置里勾选「播放失败时暂停」，避免IP被封

### 下载地址

- [Android](https://github.com/maotoumao/MusicFree/releases)
- [Windows/MacOS](https://github.com/maotoumao/MusicFreeDesktop/releases)

## Listen1

一款开源免费的跨平台音乐聚合工具，口号是"One for all free music in China"。

### 核心特点

- **多平台整合**：支持网易云、虾米、QQ音乐、酷狗、酷我、B站、咪咕
- **统一歌单**：可将不同平台的歌曲添加到同一列表
- **歌单导入**：支持导入其他平台的歌单链接
- **全平台支持**：Windows、macOS、Linux、Android、Chrome插件、Firefox插件

### 平台支持

- Windows 桌面版
- macOS 桌面版
- Linux 桌面版
- Android 安卓版
- Chrome 浏览器插件
- Firefox 浏览器插件

## 注意事项

### 关于音源

- 音质最高支持 320k
- 不支持数字专辑，反复请求可能会导致 **封禁IP**
- 仅供在线试听，禁止批量下载，批量下载会被 **封禁IP**
- 尽量避免频繁切换歌曲，否则将导致 **封禁IP**

### 关于解封

- 如果被封禁 IP，在数据流量情况下开启飞行模式切换 IP

### 使用建议

1. 这些工具仅供个人学习使用
2. 请尊重版权，支持正版音乐
3. 部分功能可能因版权政策变化而失效
4. 使用洛雪音乐时建议选择「六音音源」
5. MusicFree 用户务必开启「播放失败时暂停」

## 常见问题

### Q: 这些软件安全吗？

A: 洛雪音乐、Listen1、MusicFree 都是开源软件，代码公开可查，无隐藏广告或恶意代码。建议从 GitHub 官方 releases 下载。

### Q: 需要付费吗？

A: 完全免费。但洛雪音乐和 MusicFree 需要手动导入音源才能使用。

### Q: 支持下载吗？

A: 洛雪音乐支持下载。Listen1 主要支持播放，不支持下载。MusicFree 支持下载。

### Q: 音源链接失效怎么办？

A: 可以关注 [keep-alive](https://github.com/Huibq/keep-alive) 项目获取最新音源，该项目会持续更新。
