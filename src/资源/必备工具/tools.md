# 系统管理类：轻量高效的设备优化工具

## 1. SquirrelDisk（磁盘分析）

- **核心功能**：通过图形化图表直观展示文件夹占用比例，快速定位 GB 级大文件、重复文件及冗余缓存。
- **突出优势**：
  - 开源免费无广告
  - 安装包仅 2MB
  - 启动速度比同类工具快 3 倍
  - 支持一键跳转文件所在位置
- **适用人群**：硬盘空间告急的学生、需定期清理办公电脑的职场人。
- **系统支持**：Windows 全版本，微软商店可直接下载。

## 2. Open Hardware Monitor（硬件监控）

- **核心功能**：实时追踪 CPU/GPU 温度、电压、风扇转速，支持设置温度预警线，数据可导出为 Excel 表格。
- **突出优势**：
  - 无需安装驱动
  - 兼容 Intel、AMD 全系列处理器及 NVIDIA/AMD 显卡
  - 桌面小窗可悬浮显示关键数据
- **适用人群**：游戏玩家、视频剪辑师等需监控硬件负载的用户。
- **合规提示**：开源项目，官网提供正版安装包，无捆绑插件。

## 3. NanaZip（压缩解压）

- **核心功能**：支持 ZIP、RAR、7z 等 20+ 格式，集成右键快速解压、加密压缩等功能。
- **突出优势**：
  - 基于 7-Zip 内核优化，解压速度提升 40%
  - 界面采用 Win11 原生设计语言
  - 无任何广告弹窗
- **适用人群**：经常处理压缩包的办公族、设计师。
- **获取渠道**：微软官方应用商店，确保正版安全。

# 效率办公类：覆盖全场景的生产力助手

## 1. OneCommander（文件管理）

- **核心功能**：双栏 + 多标签页管理文件，支持颜色标记、批量重命名、预览 PDF / 图片等高级操作。
- **突出优势**：
  - 免费版已满足日常需求
  - 暗色主题可减少视觉疲劳
  - 支持将常用文件夹固定为侧边栏快捷入口
- **适用人群**：需管理大量项目文件的程序员、内容创作者。
- **系统支持**：Windows 10/11，官网提供正版安装包。

## 2. Freeplane（思维导图）

- **核心功能**：创建层级化思维导图，支持节点链接、任务分配、日程提醒，可导入 Word/Excel 数据。
- **突出优势**：
  - 完全开源
  - 插件生态丰富（如支持插入 LaTeX 公式、生成甘特图）
  - 本地存储保障数据隐私
- **适用人群**：学生（整理笔记）、项目管理者（规划流程）。
- **跨平台支持**：Windows/macOS/Linux（需预装 Java 环境）。

## 3. Obsidian（知识管理）

- **核心功能**：以「双向链接」构建个人知识网络，支持 Markdown 编辑、本地文件存储，可搭配插件扩展功能。
- **突出优势**：
  - 免费无广告
  - 数据完全归用户所有
  - 与 Raindrop.io 等工具可通过插件同步，适合搭建私人知识库
- **适用人群**：研究者、自媒体人等需深度整合信息的用户。
- **合规提示**：官方渠道下载，避免第三方修改版带来的安全风险。

# 聚合工具类：一软多用的全能帮手

## 1. JamTools（多功能工具箱）

- **核心功能**：整合 8 大高频功能 —— 截屏标注（含 OCR 文字提取）、录屏（支持自定义帧率）、格式转换（音视频 / 图片）、键鼠自动化脚本录制等。
- **突出优势**：
  - 轻量级（安装包 < 100MB）
  - 跨平台支持（Windows/macOS/Ubuntu）
  - 重复性操作可通过脚本自动执行
- **适用场景**：制作教程（录屏 + 标注）、数据录入（自动化脚本）、文件处理（格式转换）。

## 2. GOG GALAXY（游戏管理）

- **核心功能**：统一管理 Steam、Epic、Xbox 等多平台游戏库，自动同步游戏时长、存档，支持离线启动非 GOG 游戏。
- **突出优势**：
  - 完全免费，无捆绑广告
  - 可自定义游戏分类标签
  - 解决多平台切换的繁琐问题
- **适用人群**：拥有多个游戏平台账号的玩家。
- **合规保障**：GOG 官方出品，与各大平台达成授权合作，安全无侵权风险。

# 工具选用避坑指南

- **优先官方渠道**：如微软商店、工具官网，避免第三方站点的修改版（可能含恶意插件）。
- **认准开源标识**：开源工具（如 `Freeplane`、`Open Hardware Monitor`）可通过社区验证安全性，且无隐藏收费。
- **警惕捆绑安装**：若下载页出现 “高速下载”“立即安装” 等诱导按钮，大概率含捆绑软件，建议选择 “普通下载”。
